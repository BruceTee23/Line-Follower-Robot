# Line-Follower-Robot
A Robot that can follow a line using L293D motor with Arduino Uno R3 and Infared LED

What you need:
1 x Arduino Uno R3
1 x Car chassis
2 x Gear motor
2 x Car tire
2 x Speed encoder
4 x Fastener
1 x Universal wheel
1 x Battery box(batteries not included)
All necessary screw and nut. Typically M3 nuts and bolts
